# Пользовательская главная страница

## Реализованные изменения

### 1. Редирект после регистрации
✅ **Изменено**: После успешной регистрации/входа пользователь перенаправляется на главную страницу (`/`) вместо страницы профиля (`/profile`)

**Измененные файлы:**
- `app/auth/signin/page.tsx` - изменен `callbackUrl` с `/profile` на `/`
- `app/auth/signup/page.tsx` - изменен `callbackUrl` с `/profile` на `/`

### 2. Хедер главной страницы
✅ **Реализовано**: Хедер с лого, сменой языка и меню аккаунта

**Компонент**: `components/Header.tsx`

**Функционал:**
- Логотип "CINEMA" (красный цвет #e50914)
- Навигация (Главная, Фильмы, Новинки, Мой список)
- Переключатель языка (Русский/Кыргызча)
- Меню пользователя:
  - Если не авторизован: кнопки "Войти" и "Регистрация"
  - Если авторизован: аватар с выпадающим меню (Профиль, Настройки, Выйти)

### 3. Карусель
✅ **Реализовано**: Полноэкранная карусель с автопрокруткой

**Компонент**: `components/Carousel.tsx`

**Функционал:**
- Автоматическая смена слайдов каждые 7 секунд
- 3 слайда с популярными фильмами
- Плавающие карточки фильмов по краям (только на больших экранах)
- Кнопка "Войти" для неавторизованных пользователей
- Индикаторы слайдов
- Кнопка прокрутки вниз

### 4. Переключатель дат и поиск с категориями
✅ **Реализовано**: Улучшенный фильтр сеансов

**Компонент**: `components/SessionFilter.tsx`

**Функционал:**
- Переключатель "Сегодня" / "Завтра" (красная кнопка для активного состояния)
- Dropdown с категориями фильмов:
  - Все категории
  - Боевик
  - Комедия
  - Драма
  - Фантастика
  - Ужасы
  - Триллер
- Поле поиска с иконкой
- Адаптивный дизайн (вертикальное расположение на мобильных)

### 5. Карточки фильмов
✅ **Реализовано**: Карточки фильмов с информацией о сеансах

**Компонент**: `components/MovieCard.tsx`

**Функционал:**
- Постер фильма (пока заглушка с эмодзи 🎬)
- Рейтинг (звездочка + число)
- Название фильма
- Жанр
- Цена (от минимальной)
- Время сеансов (первые 3 + счетчик остальных)
- Hover эффект с предложением авторизоваться

**Компонент**: `components/MovieRow.tsx`

**Функционал:**
- Горизонтальная прокрутка фильмов
- Стрелки навигации (появляются при наведении)
- Адаптивная ширина карточек

### 6. Интеграция с базой данных
✅ **Реализовано**: Главная страница получает данные из PostgreSQL

**Файл**: `app/page.tsx`

**Функционал:**
- Загрузка фильмов со статусом `now_showing`
- Загрузка сеансов (сегодня и позже)
- Получение переводов на текущем языке
- Получение жанров с переводами
- Расчет минимальной цены
- Форматирование времени сеансов
- Разделение фильмов по категориям:
  - Популярные сегодня (первые 6)
  - Скоро в прокате (следующие 6)
  - Боевики (фильтр по жанру)

### 7. Многоязычность
✅ **Реализовано**: Поддержка русского и кыргызского языков

**Файлы переводов:**
- `app/i18n/locales/ru/landing.json`
- `app/i18n/locales/kg/landing.json`

**Добавлены переводы для:**
- Категорий фильмов
- Кнопок фильтра
- Поиска
- Карточек фильмов

## Стиль дизайна

### Цветовая схема
- **Основной фон**: Черный (`#000000`)
- **Акцентный цвет**: Красный (`#e50914`) - для кнопок, логотипа
- **Фон элементов**: Темно-серый (`#1a1a1a`, `#2a2a2a`)
- **Границы**: Темно-серые (`#2a2a2a`, `#3a3a3a`)
- **Текст**: Белый, серый (`#gray-300`, `#gray-400`, `#gray-500`)

### Типографика
- **Заголовки**: Bold, крупные размеры
- **Текст**: Regular, средние размеры
- **Кнопки**: Semibold

### Эффекты
- Плавные переходы (`transition-all`, `transition-colors`)
- Hover эффекты (изменение фона, масштабирование)
- Скругленные углы (`rounded-lg`, `rounded-full`)
- Тени (`shadow-xl`, `shadow-2xl`)

## Технологии

- **Next.js 15** - React фреймворк
- **TypeScript** - типизация
- **Prisma** - ORM для работы с PostgreSQL
- **NextAuth.js** - аутентификация
- **Tailwind CSS** - стилизация
- **Lucide React** - иконки

## Запуск проекта

```bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Открыть в браузере
http://localhost:3000
```

## База данных

Убедитесь, что в базе данных есть:
- Фильмы со статусом `now_showing`
- Сеансы с датой >= сегодня
- Переводы фильмов на русском и кыргызском
- Жанры с переводами

Для добавления тестовых данных используйте seed скрипт:
```bash
npx prisma db seed
```
