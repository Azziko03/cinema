### 2. Исправлено отображение фильмов ✅

**Проблема:** Фильмы не отображались из-за фильтрации по дате сеансов

**Решение:** Убрана фильтрация по дате, теперь показываются все фильмы со статусом `now_showing`

#### Что изменилось:

**Было:**
```typescript
where: {
  status: 'now_showing',
  sessions: {
    some: {
      startTime: { gte: today }  // ❌ Фильтровало фильмы
    }
  }
}
```

**Стало:**
```typescript
where: {
  status: 'now_showing'  // ✅ Показывает все фильмы
}
```

## 🗄️ Отображение фильмов

### Логика загрузки

```typescript
// 1. Получаем ВСЕ фильмы со статусом now_showing
const movies = await prisma.movie.findMany({
  where: {
    status: 'now_showing'  // Без фильтрации по дате!
  },
  include: {
    translations: true,
    metadata: true,
    genres: { ... },
    sessions: { ... }
  }
})

// 2. Форматируем для отображения
const formattedMovies = formatMoviesForDisplay(movies, locale)

// 3. Разделяем по категориям
const popularMovies = formattedMovies.slice(0, 6)
const comingSoonMovies = formattedMovies.slice(6, 12)
const actionMovies = formattedMovies.filter(...)
```

### Обработка сеансов

```typescript
// Если есть сеансы - показываем время
times: movie.sessions.map(s => formatTime(s.startTime))

// Если нет сеансов - показываем заглушку
times: ['Нет сеансов']
```
