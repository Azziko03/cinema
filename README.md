# 🎬 Cinema - Современный сайт кинотеатра

Полнофункциональный веб-сайт кинотеатра с онлайн-бронированием билетов, мультиязычностью и интеграцией платежной системы Finik.

## ✨ Основные возможности

- 🎥 **Афиша фильмов** - актуальная информация о фильмах с постерами, трейлерами и описаниями
- 📅 **Расписание сеансов** - удобный просмотр расписания по датам и залам
- 🎫 **Онлайн-бронирование** - интерактивная схема зала и выбор мест
- 💳 **Онлайн-оплата** - интеграция с платежной системой Finik
- 📱 **Электронные билеты** - QR-коды для быстрого входа
- 🌐 **Мультиязычность** - поддержка кыргызского, русского и английского языков
- 🤖 **AI-чат** - помощник для пользователей
- 👤 **Личный кабинет** - история заказов и управление профилем
- 🔐 **Админ-панель** - управление фильмами, расписанием и заказами с 2FA через Telegram
- 🛡️ **Разделение доступа** - строгая изоляция админов и пользователей
- 📱 **Адаптивный дизайн** - полная поддержка мобильных устройств с бургер-меню и оптимизированной каруселью

## 📱 Мобильная версия

### Для авторизованных пользователей:
- **Бургер-меню** - расширенное боковое меню с основной навигацией (Главная, Фильмы, Новинки, Мой список) и дополнительными разделами (Популярное, Расписание, Топ рейтинг, Скоро)
- **Без карусели** - для авторизованных пользователей карусель не отображается, контент начинается сразу с фильтров
- **Адаптивная навигация** - удобное управление на любом устройстве

### Для неавторизованных пользователей:
- **Полноэкранная карусель** - эффектная презентация фильмов с описаниями
- **Навигация в хедере** - классическая горизонтальная навигация на десктопе

## 🎫 Система бронирования билетов

### Функциональность:

1. **Выбор сеанса** - на странице фильма пользователь выбирает дату и время сеанса
2. **Интерактивная схема зала** - после нажатия "Выбрать места" открывается страница с:
   - Визуализацией экрана с эффектом перспективы
   - Интерактивной картой зала (идентична админ-панели)
   - Возможностью выбора нескольких мест одновременно
   - Автоматическим подсчетом стоимости (обычные + VIP места)
   - Отображением уже забронированных мест

3. **Информация о сеансе** - боковая панель с:
   - Постером фильма
   - Датой и временем сеанса
   - Названием зала
   - Ценами на обычные и VIP места
   - Списком выбранных мест с итоговой суммой

4. **Кнопка "Купить"** - активируется после выбора мест (переход на оплату - в разработке)

### Технические детали:

- **Маршрут:** `/booking/[sessionId]`
- **Компоненты:**
  - `page.tsx` - серверный компонент с загрузкой данных
  - `BookingClient.tsx` - клиентский компонент с интерактивностью
  - `not-found.tsx` - страница ошибки для несуществующих сеансов

- **Защита:**
  - Проверка авторизации (редирект на /auth/signup)
  - Проверка существования сеанса
  - Проверка, что сеанс еще не начался
  - Отображение уже забронированных мест
