# Руководство по выбору дат для сеансов

## Обзор

Реализована гибкая система выбора дат с возможностью исключения ненужных дней (например, выходных или будних дней).

---

## Как это работает

### Шаг 1: Указание диапазона дат

Админ указывает:
- **Дата начала** (обязательно)
- **Дата окончания** (опционально)

```
┌─────────────────────────────────────┐
│ Дата начала: [15.05.2026]          │
│ Дата окончания: [21.05.2026]       │
└─────────────────────────────────────┘
```

### Шаг 2: Автоматическая генерация дат

После указания диапазона автоматически генерируются все даты:

```
15.05.2026 - 21.05.2026 = 7 дней
```

### Шаг 3: Выбор конкретных дат

Появляется блок с возможностью выбора/исключения дат:

```
┌─────────────────────────────────────────────────┐
│ Выберите даты для создания сеансов              │
│ Выбрано: 5 из 7                                 │
├─────────────────────────────────────────────────┤
│ [Выбрать все] [Снять все]                      │
│ [Только будни] [Только выходные]               │
├─────────────────────────────────────────────────┤
│                                                 │
│ ☑ 15.05  чт    ☑ 16.05  пт    ☐ 17.05  сб     │
│ ☐ 18.05  вс    ☑ 19.05  пн    ☑ 20.05  вт     │
│ ☑ 21.05  ср                                    │
│                                                 │
└─────────────────────────────────────────────────┘
```

---

## Быстрые действия

### 1. Выбрать все
Выбирает все даты в диапазоне.

**Пример:**
```
15.05 - 21.05 (7 дней)
→ Все 7 дней выбраны
```

### 2. Снять все
Снимает выбор со всех дат.

**Пример:**
```
Все даты выбраны
→ Ни одна дата не выбрана
```

### 3. Только будни
Выбирает только будние дни (понедельник - пятница).

**Пример:**
```
15.05 (чт) ✓
16.05 (пт) ✓
17.05 (сб) ✗
18.05 (вс) ✗
19.05 (пн) ✓
20.05 (вт) ✓
21.05 (ср) ✓

Результат: 5 дней выбрано
```

### 4. Только выходные
Выбирает только выходные дни (суббота, воскресенье).

**Пример:**
```
15.05 (чт) ✗
16.05 (пт) ✗
17.05 (сб) ✓
18.05 (вс) ✓
19.05 (пн) ✗
20.05 (вт) ✗
21.05 (ср) ✗

Результат: 2 дня выбрано
```

**Выбранная дата:**

```
┌─────────────────┐
│ ☐ 16.05        │  ← Серая рамка

```

```
│ ☑ 17.05        │
```

## Расчет количества сеансов

### Формула:
```
Количество сеансов = 
  Выбранные даты × Временные слоты × Залы
```

### Примеры:

**Пример 1:**
```
Даты: 5 (будни)
Время: 3 (14:00, 17:00, 20:00)
Залы: 2 (Зал 1, Зал 2)

Итого: 5 × 3 × 2 = 30 сеансов
```

**Пример 2:**
```
Даты: 2 (выходные)
Время: 4 (12:00, 15:00, 18:00, 21:00)
Залы: 3 (Зал 1, Зал 2, Зал 3)

Итого: 2 × 4 × 3 = 24 сеанса
```

**Пример 3:**
```
Даты: 1 (один день)
Время: 5 (10:00, 13:00, 16:00, 19:00, 22:00)
Залы: 1 (Зал 1)

Итого: 1 × 5 × 1 = 5 сеансов
```

## Технические детали

### Генерация дат

```typescript
const generateDates = (start: string, end: string) => {
  const dates: Date[] = [];
  const startDate = new Date(start);
  const endDate = end ? new Date(end) : new Date(start);

  for (let d = new Date(startDate); d <= endDate; d.setDate(d.getDate() + 1)) {
    dates.push(new Date(d));
  }

  setGeneratedDates(dates);
  // По умолчанию все даты выбраны
  setSelectedDates(dates.map(d => d.toISOString().split('T')[0]));
};
```

### Определение будних дней

```typescript
const selectWeekdays = () => {
  const weekdays = generatedDates.filter(d => {
    const day = d.getDay();
    return day !== 0 && day !== 6; // Не воскресенье и не суббота
  });
  setSelectedDates(weekdays.map(d => d.toISOString().split('T')[0]));
};
```

### Определение выходных

```typescript
const selectWeekends = () => {
  const weekends = generatedDates.filter(d => {
    const day = d.getDay();
    return day === 0 || day === 6; // Воскресенье или суббота
  });
  setSelectedDates(weekends.map(d => d.toISOString().split('T')[0]));
};
```

### Переключение выбора даты

```typescript
const toggleDate = (dateStr: string) => {
  setSelectedDates(prev => 
    prev.includes(dateStr)
      ? prev.filter(d => d !== dateStr)  // Убрать
      : [...prev, dateStr]                // Добавить
  );
};
```

## Преимущества

### 1. Гибкость ✓
- Можно выбрать любые даты
- Можно исключить конкретные дни
- Быстрые действия для типичных сценариев

### 2. Удобство ✓
- Визуальное отображение всех дат
- Цветовое кодирование выходных
- Счетчик выбранных дат

### 3. Контроль ✓
- Полный контроль над каждой датой
- Предотвращение ошибок (валидация)
- Понятная обратная связь

### 4. Эффективность ✓
- Быстрые кнопки для типичных сценариев
- Не нужно вручную выбирать каждую дату
- Экономия времени администратора

---

## Workflow

```
1. Указать дату начала
         ↓
2. Указать дату окончания (опционально)
         ↓
3. Автоматическая генерация дат
         ↓
4. Появляется блок выбора дат
         ↓
    ┌────┴────┐
    │         │
    ▼         ▼
Быстрые    Ручной
действия   выбор
    │         │
    └────┬────┘
         ↓
5. Выбранные даты используются
   для проверки доступности
         ↓
6. Создание сеансов только
   для выбранных дат
```
