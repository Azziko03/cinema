# Страница деталей фильма

## Описание
Страница деталей фильма с возможностью просмотра информации, оценки и покупки билетов.

## Структура страницы

### 1. Основная информация (Hero Section)
- **Постер** (слева) - изображение фильма
- **Информация** (справа):
  - Название фильма (большой текст)
  - Длительность
  - Возрастные ограничения
  - Страна
  - Год выпуска
  - Дата выхода
  - Жанр

### 2. Трейлер
- Заголовок "Смотреть трейлер"
- Видео-плеер с трейлером
- Превью с кнопкой воспроизведения

### 3. Описание
- Заголовок "Описание"
- Полное описание фильма

### 4. Рейтинг
- Средний рейтинг (8.9 из 10)
- Количество отзывов
- Визуализация распределения оценок (1-10)
- Внешние рейтинги (IMDb, Кинопоиск)
- Возможность оценить фильм (1-10)
- Требуется авторизация для оценки

### 5. Покупка билета
- Выбор даты сеанса
- Выбор времени и зала
- Информация о цене
- Кнопка "Выбрать места"
- Требуется авторизация для покупки

## Файлы

### Основные компоненты
- `app/movies/[id]/page.tsx` - серверный компонент страницы
- `app/movies/[id]/MovieDetailClient.tsx` - клиентский компонент
- `app/movies/[id]/RatingSection.tsx` - секция рейтинга
- `app/movies/[id]/BuyTicketSection.tsx` - секция покупки билетов
- `app/movies/[id]/not-found.tsx` - страница 404

### API
- `app/api/movies/[id]/rating/route.ts` - API для рейтингов
  - GET - получить рейтинг фильма
  - POST - оценить фильм

### База данных
- Добавлена таблица `movie_ratings`:
  - `id` - UUID
  - `movieId` - UUID фильма
  - `userId` - UUID пользователя
  - `rating` - оценка (1-10)
  - `createdAt` - дата создания
  - `updatedAt` - дата обновления
  - Уникальный индекс на `(movieId, userId)`

## Навигация
- Клик по карточке фильма на главной странице ведет на `/movies/[id]`
- Компонент `MovieCard` обновлен для поддержки навигации

## Авторизация
- Неавторизованные пользователи могут просматривать информацию
- Для оценки фильма требуется авторизация
- Для покупки билетов требуется авторизация
- При попытке действия без авторизации - редирект на `/auth/signup`

## Адаптивность
- Мобильная версия: вертикальная компоновка
- Десктоп: горизонтальная компоновка с постером слева
- Все элементы адаптированы под разные размеры экрана

## Стилизация
- Темная тема (черный фон)
- Акцентный цвет: #e50914 (красный Netflix)
- Плавные переходы и анимации
- Hover-эффекты на интерактивных элементах
