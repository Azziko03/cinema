# Обновление мобильного дизайна страницы деталей фильма

## Изменения в дизайне

### Мобильная версия (< 1024px)

#### 1. Hero Section
- **Постер на весь экран** (500px высота)
- Градиент снизу для плавного перехода
- Название фильма поверх постера внизу
- Длительность отображается как "Duration · X minutes"

#### 2. Метаданные
- Компактное отображение:
  - Director (режиссер)
  - Shown Languages (языки показа с бейджем возрастного рейтинга)
  - Genre (жанры через запятую)
- Минималистичный стиль с серым текстом

#### 3. Watch Trailer
- Заголовок "Watch Trailer"
- Превью с кнопкой воспроизведения
- Круглая красная кнопка Play по центру

#### 4. Synopsis
- Заголовок "Synopsis"
- Описание фильма с отступами
- Компактный текст (text-sm)

#### 5. Rating Section
**Горизонтальная компоновка:**
- **Слева:** Большой рейтинг (8.9) + звезды + количество отзывов
- **Справа:** Вертикальные бары распределения (1-10)
  - Высота 80px
  - Красные бары (#e50914)
  - Цифры снизу (10px размер)

**Кнопки оценки:**
- 10 кнопок в ряд (размер 32x32px)
- Компактное расположение
- Активная кнопка: красный фон + увеличение

#### 6. Book Now Button
- Большая красная кнопка внизу
- Полная ширина с отступами
- Закругленная (rounded-full)
- Текст "Book Now"

### Desktop версия (≥ 1024px)
- Сохранен оригинальный дизайн
- Постер слева, информация справа
- Горизонтальные бары рейтинга
- Секция выбора сеансов

## Технические детали

### Адаптивность
```tsx
// Мобильная версия
<div className="lg:hidden">
  {/* Мобильный контент */}
</div>

// Desktop версия
<div className="hidden lg:block">
  {/* Desktop контент */}
</div>
```

### Цвета
- Фон: `#0a0a0a` (более темный черный)
- Акцент: `#e50914` (красный)
- Текст: белый и оттенки серого
- Бары рейтинга: красные с серым фоном

### Компоненты
1. **MovieDetailClient.tsx** - основной компонент с разделением на mobile/desktop
2. **RatingSection.tsx** - адаптивная секция рейтинга
3. **BuyTicketSection.tsx** - скрыта на мобильной версии (только кнопка Book Now)

## Особенности реализации

### Распределение рейтингов (Mobile)
```tsx
// Вертикальные бары с процентным заполнением снизу вверх
<div style={{ 
  height: `${percentage}%`,
  marginTop: `${100 - percentage}%`
}} />
```

### Градиент на постере
```tsx
<div className="absolute inset-0 bg-gradient-to-b from-transparent via-transparent to-[#0a0a0a]" />
```

### Компактные кнопки рейтинга
- Размер: 32x32px (mobile) vs 48x48px (desktop)
- Отступы: 6px (mobile) vs 8px (desktop)
- Шрифт: text-sm (mobile) vs text-lg (desktop)
