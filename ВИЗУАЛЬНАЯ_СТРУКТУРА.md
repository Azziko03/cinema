## 🎬 Анимации

### Карусель
- **Transition**: 7 секунд
- **Effect**: fade + slide
- **Easing**: ease-in-out

### Scroll Down
- **Animation**: bounce
- **Duration**: infinite
- **Delay**: 0s

### Hover
- **Duration**: 150-300ms
- **Properties**: background, transform, opacity
- **Easing**: ease-in-out

## ✨ Особые эффекты

### Backdrop Blur
```css
backdrop-blur-md    /* Плавающие карточки */
backdrop-blur-sm    /* Dropdown меню */
```

### Градиенты
```css
/* Хедер */
bg-gradient-to-b from-black to-transparent

/* Стрелки навигации */
bg-gradient-to-r from-black to-transparent
bg-gradient-to-l from-black to-transparent

/* Низ карусели */
bg-gradient-to-t from-black to-transparent
```

### Тени
```css
shadow-xl     /* Dropdown */
shadow-2xl    /* Карточки, модалы */
```

## 🎯 Ключевые точки

1. **Фиксированный хедер** - всегда виден
2. **Полноэкранная карусель** - первый экран
3. **Центрированный фильтр** - max-width: 1536px
4. **Горизонтальная прокрутка** - для рядов фильмов
5. **Адаптивная раскладка** - от мобильных до десктопа

---

**Все элементы следуют единому стилю без использования внешних UI библиотек!** 🎨
