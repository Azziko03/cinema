# 🔐 Админ-панель с 2FA через Telegram

## Что это?

Полнофункциональная админ-панель с двухфакторной аутентификацией через Telegram Bot. Обеспечивает строгое разделение доступа между администраторами и обычными пользователями.

## Ключевые особенности

- 🔐 **2FA через Telegram** - дополнительный уровень безопасности
- 🛡️ **Разделение ролей** - админы и пользователи изолированы друг от друга
- 🌐 **Мультиязычность** - поддержка русского и кыргызского языков
- 🎨 **Современный UI** - красивый дизайн с градиентами и анимациями
- ⚡ **Быстрая настройка** - 5 минут до первого входа

## Быстрый старт

### 1. Создайте Telegram бота

```
1. Откройте @BotFather в Telegram
2. Отправьте /newbot
3. Сохраните токен
```

### 2. Получите ваш Telegram User ID

```
1. Откройте @userinfobot в Telegram
2. Отправьте любое сообщение
3. Сохраните User ID
```

### 3. Добавьте настройки в БД

```bash
npm run db:studio
```

В таблице `Setting` заполните:
- `ADMIN_TELEGRAM_BOT_TOKEN` → ваш токен бота
- `ADMIN_TELEGRAM_USER_ID` → ваш User ID

### 4. Войдите

```
URL: http://localhost:3000/admin/login
Email: admin@cinema.kg
Пароль: admin123
```

## Как это работает

```
┌─────────────────────────────────────────────────────────────┐
│                    ПРОЦЕСС ВХОДА                            │
└─────────────────────────────────────────────────────────────┘

1. Админ открывает /admin/login
   ↓
2. Вводит email и пароль
   ↓
3. Система проверяет:
   ✓ Email существует
   ✓ Пароль правильный
   ✓ Роль = admin
   ✓ Статус = active
   ↓
4. Генерируется 6-значный код
   ↓
5. Код отправляется в Telegram
   ↓
6. Админ вводит код
   ↓
7. Система проверяет:
   ✓ Код правильный
   ✓ Код не истек (5 мин)
   ↓
8. Авторизация через NextAuth
   ↓
9. Redirect на /admin/dashboard
```

## Разделение доступа

### Администраторы (role: admin)

✅ **Могут:**
- Заходить на `/admin/*`
- Управлять контентом
- Просматривать статистику

❌ **НЕ могут:**
- Заходить на `/profile`
- Заходить на `/auth/signin`
- Видеть пользовательские страницы

### Пользователи (role: user)

✅ **Могут:**
- Заходить на `/profile`
- Бронировать билеты
- Просматривать историю заказов

❌ **НЕ могут:**
- Заходить на `/admin/*`
- Видеть админские страницы
- Управлять контентом

## API Endpoints

### POST `/api/admin/login`

Первый шаг входа - проверка email и пароля.

**Request:**
```json
{
  "email": "admin@cinema.kg",
  "password": "admin123"
}
```

**Response:**
```json
{
  "message": "Код отправлен в Telegram",
  "userId": "uuid",
  "requiresTwoFactor": true
}
```

### POST `/api/admin/verify-code`

Второй шаг входа - проверка кода из Telegram.

**Request:**
```json
{
  "userId": "uuid",
  "code": "123456"
}
```

**Response:**
```json
{
  "success": true,
  "message": "Код подтвержден",
  "user": {
    "id": "uuid",
    "email": "admin@cinema.kg",
    "name": "Администратор",
    "role": "admin"
  }
}
```

## Troubleshooting

### Код не приходит

1. Убедитесь, что начали диалог с ботом (`/start`)
2. Проверьте настройки в таблице `settings`
3. Проверьте логи сервера

### Ошибка "Доступ запрещен"

1. Проверьте роль пользователя (`role = "admin"`)
2. Проверьте статус (`status = "active"`)
3. Проверьте email verification

### Код истек

Коды действительны 5 минут. Нажмите "Отправить код повторно".
