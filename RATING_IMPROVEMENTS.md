# Улучшения системы рейтинга

## Изменения

### 1. Компактный постер с информацией в ряд (Mobile)
**Было:** Постер на весь экран с информацией снизу  
**Стало:** 
- Маленький постер слева (128px ширина)
- Информация справа в одном блоке
- Одинаковая высота обоих элементов (192px)
- Компактное отображение метаданных

### 2. Русский язык
**Изменено:**
- "Watch Trailer" → "Смотреть трейлер"
- "Synopsis" → "Описание"
- "Rating" → "Рейтинг"
- "Book Now" → "Забронировать"
- "Rate this Movie" → "Оцените фильм"
- "Duration" → "Длительность"
- "Director" → "Режиссер"
- "Genre" → "Жанр"

### 3. Кнопка "Оценить" после выбора рейтинга
**Новая логика:**
1. Пользователь выбирает оценку (1-10)
2. Появляется кнопка "Оценить"
3. После клика оценка отправляется в БД
4. Кнопки блокируются после оценки
5. Отображается "Ваша оценка: X"

**Код:**
```tsx
{selectedRating && !userRating && (
  <button onClick={handleSubmitRating}>
    Оценить
  </button>
)}
```

### 4. Реальный рейтинг из БД
**Подключено:**
- ✅ Средний рейтинг вычисляется из всех оценок
- ✅ Количество голосов (totalVotes) из БД
- ✅ Отображение: "X голос/голоса/голосов"
- ✅ Если нет оценок: "Нет оценок" и "—"

**API изменения:**
```typescript
// GET /api/movies/[id]/rating
{
  averageRating: 8.5,
  totalVotes: 42,
  userRating: 9
}
```

### 5. Звезды из 5 (не из 10)
**Конвертация:**
```typescript
const starRating = averageRating / 2
// 10 → 5 звезд
// 8 → 4 звезды
// 5 → 2.5 звезды
```

### 6. Упрощенная статистика
**Было:** 
- Сложное распределение с барами 1-10
- Много визуальных элементов

**Стало:**
- **Mobile:** Центрированный рейтинг + звезды + количество голосов
- **Desktop:** Слева - рейтинг со звездами, справа - простая статистика "X/10"
- Внешние рейтинги (IMDb, Кинопоиск) отдельным блоком

## База данных

### Таблица movie_ratings
```sql
CREATE TABLE movie_ratings (
  id UUID PRIMARY KEY,
  movie_id UUID REFERENCES movies(id),
  user_id UUID REFERENCES users(id),
  rating INTEGER CHECK (rating >= 1 AND rating <= 10),
  created_at TIMESTAMP,
  updated_at TIMESTAMP,
  UNIQUE(movie_id, user_id)
);
```

### Вычисление среднего
```typescript
const averageRating = ratings.length > 0
  ? ratings.reduce((sum, r) => sum + r.rating, 0) / ratings.length
  : 0
```

## Особенности

1. **Защита от повторной оценки:** Один пользователь = одна оценка
2. **Блокировка после оценки:** Кнопки становятся неактивными
3. **Реальное время:** Рейтинг обновляется сразу после оценки
4. **Авторизация:** Требуется вход для оценки
5. **Склонение:** "1 голос", "2 голоса", "5 голосов"
