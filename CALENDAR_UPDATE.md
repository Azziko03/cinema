# Обновление календаря в странице создания сеанса

## Что изменилось

### Было:
- Простой список из 60 дней подряд
- Нельзя было выбрать конкретный месяц или год
- Показывались только следующие 60 дней от текущей даты

### Стало:
- **Полноценный календарь** с навигацией по месяцам и годам
- **Выбор месяца** через выпадающий список (все 12 месяцев)
- **Выбор года** через выпадающий список (текущий год + 4 года вперед)
- **Стрелки навигации** для быстрого переключения между месяцами
- **Кнопка "Сегодня"** для быстрого возврата к текущей дате
- **Календарная сетка** с днями недели (Пн-Вс)
- **Блокировка прошедших дат** - нельзя выбрать даты в прошлом

## Новые возможности

### 1. Навигация по месяцам
- **Стрелка влево (←)**: Переход к предыдущему месяцу
- **Стрелка вправо (→)**: Переход к следующему месяцу
- При переходе с декабря на январь автоматически меняется год

### 2. Выбор месяца
- Дропдаун со всеми 12 месяцами года
- Названия на русском языке
- Мгновенное обновление календаря при выборе

### 3. Выбор года
- Дропдаун с годами (текущий + 4 года вперед)
- Позволяет планировать сеансы на будущее
- Мгновенное обновление календаря при выборе

### 4. Кнопка "Сегодня"
- Быстрый возврат к текущему месяцу и году
- Удобно после навигации по будущим датам

### 5. Улучшенный дизайн
- Заголовок календаря с названием месяца и года
- Строка с днями недели (Пн-Вс)
- Календарная сетка 7x6 (максимум)
- Пустые ячейки для дней предыдущего месяца

### 6. Визуальные индикаторы
- **Текущий день**: Синяя двойная рамка
- **Выбранный день**: Красный фон (#e50914) с тенью
- **Прошедшие дни**: Темно-серые, неактивны
- **Занятые дни**: Серые, неактивны
- **Доступные дни**: Белый текст, можно выбрать

### Новые функции

#### `generateCalendarDays()`
Генерирует дни для выбранного месяца и года:
- Вычисляет первый и последний день месяца
- Добавляет пустые ячейки для выравнивания по дням недели
- Возвращает массив дат (или null для пустых ячеек)

#### `goToPreviousMonth()`
Переход к предыдущему месяцу:
- Если январь → переход к декабрю предыдущего года
- Иначе → уменьшение месяца на 1

#### `goToNextMonth()`
Переход к следующему месяцу:
- Если декабрь → переход к январю следующего года
- Иначе → увеличение месяца на 1

#### `goToToday()`
Возврат к текущей дате:
- Устанавливает текущий месяц
- Устанавливает текущий год

#### `isPastDate(date: Date)`
Проверка, является ли дата прошедшей:
- Сравнивает с текущей датой (без учета времени)
- Возвращает true для прошедших дат

## Преимущества

1. **Удобная навигация**: Легко переключаться между месяцами и годами
2. **Планирование на будущее**: Можно создавать сеансы на несколько лет вперед
3. **Защита от ошибок**: Нельзя выбрать прошедшие даты
4. **Интуитивный интерфейс**: Привычный вид календаря
5. **Быстрый доступ**: Кнопка "Сегодня" для возврата к текущей дате
6. **Визуальная ясность**: Четкое отображение дней недели и структуры месяца

## Примеры использования

### Создание сеанса на следующий месяц
1. Нажмите стрелку **→** для перехода к следующему месяцу
2. Выберите нужную дату
3. Продолжите создание сеанса

### Создание сеанса через полгода
1. Откройте дропдаун **"Месяц"**
2. Выберите нужный месяц (например, Июль)
3. Если нужно, измените год в дропдауне **"Год"**
4. Выберите нужную дату
5. Продолжите создание сеанса

### Возврат к текущей дате
1. После навигации по будущим месяцам
2. Нажмите кнопку **"Сегодня"**
3. Календарь вернется к текущему месяцу
