## 1️⃣ Header.tsx

### Описание
Фиксированный хедер с логотипом, навигацией, переключателем языка и меню пользователя.

### Особенности
- Фиксированная позиция (fixed top)
- Градиентный фон (от черного к прозрачному)
- Адаптивная навигация (скрывается на мобильных)
- Интеграция с LanguageSwitcher и UserMenu

### Использование
```tsx
import Header from '@/components/Header'

<Header translations={translations} locale={locale} />
```

### Стиль
- Фон: `bg-gradient-to-b from-black to-transparent`
- Высота: `h-16`
- Логотип: красный `text-[#e50914]`

---

## 2️⃣ Carousel.tsx

### Описание
Полноэкранная карусель с автоматической сменой слайдов и плавающими карточками.

### Особенности
- Автосмена каждые 7 секунд
- 3 слайда с информацией о фильмах
- Плавающие карточки по краям (только на lg+)
- Индикаторы слайдов
- Кнопка прокрутки вниз с анимацией bounce

### Использование
```tsx
import Carousel from '@/components/Carousel'

<Carousel translations={translations} />
```

### Стиль
- Высота: `h-screen` (100vh)
- Фон: чистый черный `bg-black`
- Плавающие карточки: `backdrop-blur-md bg-white/5`

## 3️⃣ SessionFilter.tsx

Фильтр для выбора даты и категории фильмов с поиском.

### Особенности
- Переключатель "Сегодня" / "Завтра"
- Dropdown с 7 категориями
- Поле поиска с иконкой
- Адаптивная раскладка (вертикальная на мобильных)

### Использование

- Активная кнопка: `bg-[#e50914]` (красная)
- Неактивная: `bg-[#1a1a1a]` (темно-серая)

## 📊 Главная страница (app/page.tsx)

### Логика загрузки данных

```typescript
// 1. Получение фильмов из БД
const movies = await prisma.movie.findMany({
  where: {
    status: 'now_showing',
    sessions: {
      some: {
        startTime: { gte: today }
      }
    }
  },
  include: {
    translations: true,
    metadata: true,
    genres: { include: { genre: { include: { translations: true } } } },
    sessions: { where: { startTime: { gte: today } }, orderBy: { startTime: 'asc' } }
  }
})

// 2. Форматирование для отображения
const formattedMovies = movies.map(movie => ({
  id: movie.id,
  title: translation?.title,
  genre: genres.join(', '),
  rating: Number(movie.metadata?.imdbRating),
  price: Math.min(...movie.sessions.map(s => Number(s.basePrice))),
  times: movie.sessions.map(s => formatTime(s.startTime))
}))

// 3. Разделение по категориям
const popularMovies = formattedMovies.slice(0, 6)
const comingSoonMovies = formattedMovies.slice(6, 12)
const actionMovies = formattedMovies.filter(m => m.genre.includes('Боевик'))
```

### Структура
```tsx
<div className="min-h-screen bg-black text-white">
  <Header translations={translations} locale={locale} />
  
  <Carousel translations={translations} />
  
  <main className="pb-12">
    <div className="container mx-auto space-y-12 md:space-y-16 py-12">
      <div className="max-w-6xl mx-auto px-4">
        <SessionFilter translations={translations} />
      </div>
      
      <MovieRow title="Популярные сегодня" movies={popularMovies} />
      <MovieRow title="Скоро в прокате" movies={comingSoonMovies} />
      <MovieRow title="Боевики" movies={actionMovies} />
    </div>
  </main>
</div>
```
