# Финальное обновление страницы создания сеанса

## Что изменилось

### 1. ❌ Убрана возможность выбора нескольких залов
**Было:** Можно было выбрать несколько залов одновременно  
**Стало:** Можно выбрать только один зал

**Причина:** Упрощение процесса создания сеансов

**Изменения:**
- Дропдаун теперь работает как single-select (одиночный выбор)
- При выборе зала дропдаун автоматически закрывается
- Убраны чекбоксы
- Убраны теги с выбранными залами
- Показывается только один выбранный зал

### 2. ✅ Добавлена возможность выбора нескольких дней
**Было:** Можно было выбрать только одну дату  
**Стало:** Можно выбрать несколько дат одновременно

**Как работает:**
- Клик на день **добавляет** его в список выбранных
- Повторный клик на день **убирает** его из списка
- Выбранные дни отмечены красным цветом и галочкой ✓
- Над календарем показывается список выбранных дат с возможностью удаления

**Пример:**
```
Выбрано дат: 3
[9 мая ×] [10 мая ×] [15 мая ×]
```

### 3. ✅ Добавлена возможность добавления нескольких временных слотов
**Было:** Для каждой даты можно было указать только одно время  
**Стало:** Для каждой даты можно добавить несколько временных слотов

**Как работает:**
- Каждая выбранная дата имеет свою секцию с временными слотами
- По умолчанию создается один пустой слот
- Кнопка **"+ Добавить время"** добавляет новый слот для этой даты
- Каждый слот можно удалить (кроме последнего)
- Время "До" автоматически вычисляется для каждого слота
- Индикатор доступности (✓ или ×) для каждого слота

**Пример для одной даты:**
```
┌─────────────────────────────────────────┐
│ Понедельник, 9 мая      [+ Добавить время] │
├─────────────────────────────────────────┤
│ 1. От: 14:00  До: 16:30  ✓  [×]        │
│ 2. От: 17:00  До: 19:30  ✓  [×]        │
│ 3. От: 20:00  До: 22:30  ✓  [×]        │
└─────────────────────────────────────────┘
```

### 4. 🔄 Независимые временные слоты для каждой даты
**Важно:** Каждая дата имеет свои собственные временные слоты

**Пример:**
- **9 мая**: 14:00, 17:00, 20:00
- **10 мая**: 15:00, 18:00, 21:00
- **15 мая**: 16:00, 19:00

При изменении даты (добавлении/удалении) временные слоты других дат не затрагиваются.

## Новая логика работы

### Процесс создания сеансов

#### Шаг 1: Выбор фильма
- Выбираете один фильм из дропдауна
- Длительность фильма используется для автоматического расчета времени окончания

#### Шаг 2: Выбор зала
- Выбираете один зал из дропдауна
- Зал будет использован для всех создаваемых сеансов

#### Шаг 3: Даты и время
1. **Выбор дат:**
   - Кликайте на дни в календаре для добавления/удаления
   - Можно выбрать любое количество дат
   - Выбранные даты отображаются над календарем

2. **Указание времени для каждой даты:**
   - Для каждой выбранной даты появляется своя секция
   - Добавляйте временные слоты кнопкой "+ Добавить время"
   - Указывайте время начала (время окончания вычисляется автоматически)
   - Система проверяет доступность каждого слота

3. **Проверка конфликтов:**
   - Для каждой даты загружаются существующие сеансы
   - Показываются занятые времена с учетом 20-минутного перерыва
   - Индикатор ✓ или × показывает доступность каждого слота

4. **Создание сеансов:**
   - Кнопка показывает общее количество сеансов
   - Создаются все сеансы для всех дат и всех временных слотов

### Пример создания сеансов

**Входные данные:**
- Фильм: "Аватар 2" (192 мин)
- Зал: "Зал 1"
- Даты: 9 мая, 10 мая, 15 мая
- Время для 9 мая: 14:00, 20:00
- Время для 10 мая: 15:00, 18:00, 21:00
- Время для 15 мая: 16:00

**Результат:**
Будет создано **6 сеансов**:
1. 9 мая, 14:00-17:12, Зал 1
2. 9 мая, 20:00-23:12, Зал 1
3. 10 мая, 15:00-18:12, Зал 1
4. 10 мая, 18:00-21:12, Зал 1
5. 10 мая, 21:00-00:12, Зал 1
6. 15 мая, 16:00-19:12, Зал 1

### Новые функции

#### `handleDateToggle(date: Date)`
Добавляет или убирает дату из списка выбранных:
- Если дата уже выбрана → убирает её и удаляет временные слоты
- Если дата не выбрана → добавляет её и создает первый временной слот

#### `addTimeSlot(dateStr: string)`
Добавляет новый временной слот для конкретной даты

#### `removeTimeSlot(dateStr: string, slotId: string)`
Удаляет временной слот (если их больше одного)

#### `updateTimeSlot(dateStr: string, slotId: string, field: 'from' | 'to', value: string)`
Обновляет значение временного слота (время "от" или "до")

#### `isTimeSlotAvailable(dateStr: string, startTime: string)`
Проверяет доступность времени для конкретной даты

## Преимущества новой системы

### 1. Гибкость
- Можно создать сеансы на любое количество дней
- Разное количество сеансов для разных дней
- Независимое управление временем для каждой даты

### 2. Удобство
- Визуальный выбор дат в календаре
- Простое добавление/удаление временных слотов
- Автоматическое вычисление времени окончания

### 3. Безопасность
- Проверка конфликтов для каждого слота
- Учет 20-минутного перерыва
- Визуальная индикация проблем

### 4. Эффективность
- Массовое создание сеансов за один раз
- Меньше кликов для создания множества сеансов
- Быстрая навигация по датам

## Примеры использования

### Пример 1: Создание сеансов на выходные
1. Выберите фильм "Человек-паук"
2. Выберите зал "Зал 1"
3. Выберите даты: Суббота (13 мая), Воскресенье (14 мая)
4. Для субботы добавьте: 12:00, 15:00, 18:00, 21:00
5. Для воскресенья добавьте: 13:00, 16:00, 19:00
6. Нажмите "Создать 7 сеансов"

### Пример 2: Создание сеансов на неделю
1. Выберите фильм "Аватар 2"
2. Выберите зал "Зал 2"
3. Выберите все дни недели (7 дней)
4. Для будних дней: 18:00, 21:00
5. Для выходных: 12:00, 15:00, 18:00, 21:00
6. Нажмите "Создать 18 сеансов"

### Пример 3: Добавление дополнительных сеансов
1. Выберите фильм
2. Выберите зал
3. Выберите дату (например, 15 мая)
4. Добавьте время: 10:00 (утренний сеанс)
5. Нажмите "Создать 1 сеанс"
