# Обновление страницы создания сеанса

## Что было изменено

### 1. Индикатор процесса (3 этапа)
В верхней части страницы добавлен визуальный индикатор процесса создания сеанса с тремя этапами:
- **Этап 1**: Выбор фильма
- **Этап 2**: Выбор залов
- **Этап 3**: Дата и время

Каждый этап отображается с номером (или галочкой при завершении) и описанием.

### 2. Выбор фильма (Этап 1)
- **Дропдаун меню** вместо сетки карточек
- При клике на дропдаун открывается список фильмов
- Каждый фильм отображается горизонтально с:
  - Маленькой картинкой (постером)
  - Названием
  - Длительностью
  - Возрастными ограничениями
  - Статусом (В прокате/Скоро/Архив)
- Выбранный фильм отображается отдельно под дропдауном

### 3. Выбор залов (Этап 2)
- **Дропдаун меню** с возможностью множественного выбора
- Можно выбрать несколько залов одновременно
- Выбранные залы отображаются в виде тегов с возможностью удаления
- Кнопки навигации: "Назад" и "Далее"

### 4. Дата и время (Этап 3)

#### Календарь
- **Автоматически отображается** (не нужно кликать на иконку)
- **Навигация по месяцам и годам**:
  - Стрелки влево/вправо для переключения месяцев
  - Дропдаун для выбора месяца
  - Дропдаун для выбора года (текущий год + 4 года вперед)
  - Кнопка "Сегодня" для быстрого возврата к текущей дате
- Показывает выбранный месяц в виде календарной сетки
- Дни отображаются в сетке 7 колонок (неделя)
- Заголовок показывает текущий месяц и год
- Дни недели (Пн-Вс) отображаются над календарем
- Каждый день показывает только число
- Текущий день выделен синей двойной рамкой
- Выбранный день выделен красным цветом (#e50914) с тенью
- Прошедшие дни неактивны (темно-серые, нельзя кликнуть)
- Полностью занятые дни неактивны (серые, нельзя кликнуть)

#### Занятые времена
При выборе даты автоматически загружаются и отображаются:
- Все существующие сеансы на эту дату
- Для каждого сеанса показывается:
  - Название фильма
  - Название зала
  - Время начала и окончания
  - **+ 20 минут перерыва** после окончания

#### Выбор времени
- **Поле "От"**: Админ вручную указывает время начала
- **Поле "До"**: Автоматически заполняется на основе:
  - Времени начала
  - Длительности выбранного фильма
- Поле "До" только для чтения (нельзя редактировать)

#### Логика перерывов (20 минут)
- Между сеансами автоматически учитывается 20-минутный перерыв
- Если сеанс заканчивается в 21:00, следующий можно начать только в 21:20
- Система автоматически проверяет доступность времени
- Если время недоступно, показывается предупреждение

#### Валидация
- Если нет ни одного сеанса на выбранную дату - можно выбрать любое время
- Если есть сеансы - система показывает занятые времена и проверяет конфликты
- Визуальная индикация:
  - ✓ Зеленый - время доступно
  - ⚠️ Красный - время недоступно (конфликт)

### 5. Дополнительные параметры
- Базовая цена (обязательно)
- Язык (Оригинал/Дубляж)
- Формат (2D/3D/IMAX)

### 6. Создание сеансов
- Кнопка "Создать N сеансов" (где N - количество выбранных залов)
- Создается по одному сеансу для каждого выбранного зала
- Все сеансы создаются с одинаковыми параметрами (время, цена, язык, формат)

## Новые API endpoints

### `/api/admin/sessions/by-date`
**POST** - Получение сеансов для конкретной даты и залов

**Параметры:**
```json
{
  "date": "2024-01-15",
  "hallIds": ["uuid1", "uuid2"]
}
```

**Ответ:**
```json
{
  "sessions": [
    {
      "id": "uuid",
      "startTime": "2024-01-15T17:00:00Z",
      "endTime": "2024-01-15T19:30:00Z",
      "movieTitle": "Название фильма",
      "hallName": "Зал 1"
    }
  ]
}
```

## Технические детали

### Константы
- `BREAK_TIME_MINUTES = 20` - Перерыв между сеансами

### Основные функции
- `loadExistingSessions()` - Загрузка существующих сеансов для даты
- `handleDateSelect()` - Обработка выбора даты
- `isTimeSlotAvailable()` - Проверка доступности времени
- `generateCalendarDays()` - Генерация дней календаря для выбранного месяца
- `goToPreviousMonth()` - Переход к предыдущему месяцу
- `goToNextMonth()` - Переход к следующему месяцу
- `goToToday()` - Возврат к текущей дате
- `isPastDate()` - Проверка, является ли дата прошедшей

### Состояния компонента
- `currentStep` - Текущий этап (1, 2 или 3)
- `selectedMovieId` - ID выбранного фильма
- `selectedHallIds` - Массив ID выбранных залов
- `selectedDate` - Выбранная дата
- `currentMonth` - Текущий отображаемый месяц (0-11)
- `currentYear` - Текущий отображаемый год
- `existingSessions` - Существующие сеансы на выбранную дату
- `timeFrom` / `timeTo` - Время начала и окончания
- `basePrice`, `language`, `format` - Дополнительные параметры

## Улучшения UX

1. **Пошаговая навигация** - Пользователь видит прогресс и может вернуться назад
2. **Автоматические вычисления** - Время окончания рассчитывается автоматически
3. **Визуальная обратная связь** - Цветовая индикация доступности времени
4. **Предотвращение ошибок** - Валидация на каждом этапе
5. **Информативность** - Показ занятых времен с учетом перерывов
6. **Множественный выбор залов** - Создание нескольких сеансов за раз
