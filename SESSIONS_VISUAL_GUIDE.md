## 🎯 Защита от скролла в поле цены

### ❌ БЫЛО:
```
Пользователь скроллит страницу
        ↓
Курсор случайно над полем "Цена"
        ↓
Цена меняется с 300 на 305 😱
```

### ✅ СТАЛО:
```
Пользователь скроллит страницу
        ↓
Курсор над полем "Цена"
        ↓
onWheel → blur() → фокус убирается
        ↓
Цена НЕ меняется ✅
```

**Код:**
```tsx
<input
  type="number"
  onWheel={(e) => e.currentTarget.blur()}
  // ... остальные пропсы
/>
```

## 🎨 Цветовая схема

### Прогресс-бары:
- 🟡 **VIP места**: `bg-yellow-500` (желтый)
- 🔵 **Обычные места**: `bg-blue-500` (синий)
- 🔴 **Всего**: `bg-[#e50914]` (красный, фирменный цвет)

### Иконки:
- ⭐ **VIP**: Звезда (star icon)
- 👤 **Обычные**: Пользователь (user icon)

### Статусы фильмов:
- 🟢 **В прокате**: `bg-green-500/20 text-green-400`
- 🔵 **Скоро**: `bg-blue-500/20 text-blue-400`
- ⚫ **Архив**: `bg-gray-500/20 text-gray-400`

## 📱 Адаптивность

Колонка "Занято мест" адаптируется под размер экрана:
- Прогресс-бары имеют `max-w-[80px]` для компактности
- Текст использует `text-xs` для экономии места
- Иконки `w-3 h-3` для баланса размера
