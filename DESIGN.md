# 🎨 Дизайн Cinema - Netflix Style

## Цветовая палитра

```css
--background: #141414      /* Основной фон */
--foreground: #ffffff      /* Основной текст */
--primary: #e50914         /* Красный акцент (как Netflix) */
--secondary: #2a2a2a       /* Вторичный фон */
--card: #181818            /* Фон карточек */
--hover: #2f2f2f           /* Hover состояние */
--border: #2a2a2a          /* Границы */
```

## Компоненты

### Header
- Фиксированный верхний хедер с градиентом
- Логотип CINEMA красного цвета
- Навигация (скрыта на мобильных)
- Поиск, уведомления, язык, профиль

### Hero Carousel
- Полноэкранный баннер (70-85vh)
- Автосмена каждые 7 секунд
- Кнопки "Воспроизвести" и "Подробнее"
- Градиентный оверлей снизу
- Индикаторы слайдов

### SessionFilter
- Кнопки "Сегодня" / "Завтра"
- Поиск с иконкой
- Минималистичный дизайн

### MovieCard
- Соотношение сторон 2:3
- Hover эффект с увеличением (scale 1.05)
- Кнопки Play и Plus при наведении
- Рейтинг со звездой
- Цена и время сеансов

### MovieRow
- Горизонтальная прокрутка
- Стрелки навигации при hover
- Плавная анимация
- Скрытый скроллбар

### BottomNav
- Фиксированная нижняя навигация
- 5 разделов с иконками
- Активное состояние

## Особенности

### Hover эффекты
```css
.movie-card:hover {
  transform: scale(1.05);
  z-index: 10;
}
```

### Градиенты
- Hero: `linear-gradient(to bottom, transparent 0%, rgba(20, 20, 20, 0.7) 50%, #141414 100%)`
- Header: `from-[#141414] to-transparent`

### Анимации
- Плавные переходы 0.3s
- Smooth scroll для рядов фильмов
- Fade эффекты для оверлеев

## Адаптивность

### Mobile (< 768px)
- 2 колонки в гриде
- Компактные карточки (150px)
- Скрытая навигация в хедере
- Нижний навбар

### Tablet (768px - 1024px)
- 3-4 колонки
- Средние карточки (200px)

### Desktop (> 1024px)
- 4-5 колонок
- Полная навигация в хедере
- Hover эффекты
- Стрелки навигации в рядах

## Типографика

- Заголовки: Bold, 2xl-6xl
- Текст: Regular, sm-base
- Цены: Bold, lg-xl
- Кнопки: Semibold, sm-base

## Иконки

Используется библиотека **lucide-react**:
- Film, Ticket, Bot, Settings, User
- Play, Plus, Info
- Search, Bell, Globe
- Star, ChevronLeft, ChevronRight
