# Инструкция: Создание сеанса

## Обзор

Страница создания сеанса теперь имеет улучшенный интерфейс с пошаговым процессом из 3 этапов.

## Процесс создания сеанса

### Этап 1: Выбор фильма

1. Откройте дропдаун "Выберите фильм"
2. Просмотрите список доступных фильмов
3. Каждый фильм показывает:
   - Постер
   - Название
   - Длительность (в минутах)
   - Возрастное ограничение
   - Статус (В прокате/Скоро/Архив)
4. Кликните на нужный фильм
5. Выбранный фильм отобразится под дропдауном
6. Нажмите кнопку **"Далее →"**

### Этап 2: Выбор залов

1. Откройте дропдаун "Выберите залы"
2. Выберите один или несколько залов (можно выбрать все)
3. Для каждого зала показывается:
   - Название зала
   - Количество мест
4. Выбранные залы отображаются в виде тегов
5. Можно удалить зал, нажав на "×"
6. Нажмите кнопку **"Далее →"** или **"← Назад"** для возврата

### Этап 3: Дата и время

#### Выбор даты

1. Календарь отображается автоматически
2. **Навигация по календарю**:
   - Используйте стрелки **←** и **→** для переключения между месяцами
   - Выберите месяц из выпадающего списка
   - Выберите год из выпадающего списка (доступны текущий год + 4 года вперед)
   - Нажмите кнопку **"Сегодня"** для быстрого возврата к текущей дате
3. Календарь показывает:
   - Название месяца и года в заголовке
   - Дни недели (Пн-Вс)
   - Все дни выбранного месяца
4. Кликните на нужную дату
5. **Особенности дней**:
   - Текущий день выделен **синей двойной рамкой**
   - Выбранный день выделен **красным цветом** с тенью
   - Прошедшие дни **темно-серые** и неактивны (нельзя выбрать)
   - Дни с полной загрузкой **серые** и неактивны

#### Просмотр занятых времен

После выбора даты автоматически загружаются существующие сеансы:

- **Если есть сеансы**: Отображается список с:
  - Названием фильма
  - Названием зала
  - Временем начала и окончания
  - **+ 20 минут перерыва** (обязательный перерыв между сеансами)

- **Если нет сеансов**: Показывается сообщение "На эту дату нет сеансов"

#### Указание времени

1. **Время начала (От)**:
   - Введите время начала сеанса вручную
   - Формат: ЧЧ:ММ (например, 17:00)

2. **Время окончания (До)**:
   - Заполняется автоматически
   - Рассчитывается: Время начала + Длительность фильма
   - Поле только для чтения

3. **Проверка доступности**:
   - ✓ Зеленое сообщение: "Время доступно для создания сеанса"
   - ⚠️ Красное сообщение: "Время недоступно. Учитывайте 20-минутный перерыв"

#### Дополнительные параметры

1. **Базовая цена** (обязательно):
   - Укажите цену билета в сомах
   - Пример: 250

2. **Язык**:
   - Оригинал (по умолчанию)
   - Дубляж

3. **Формат**:
   - 2D (по умолчанию)
   - 3D
   - IMAX

#### Создание сеансов

1. Проверьте все параметры
2. Нажмите кнопку **"Создать N сеансов"** (где N - количество выбранных залов)
3. Система создаст по одному сеансу для каждого выбранного зала
4. После успешного создания вы будете перенаправлены на страницу списка сеансов

## Важные правила

### Перерыв между сеансами (20 минут)

- Между сеансами автоматически учитывается 20-минутный перерыв
- Это время необходимо для уборки зала и подготовки к следующему сеансу

**Пример:**
- Сеанс 1: 17:00 - 19:30
- Перерыв: 19:30 - 19:50 (20 минут)
- Сеанс 2: Можно начать с 19:50

### Проверка конфликтов

Система автоматически проверяет:
- Пересечение времени с существующими сеансами
- Учет 20-минутного перерыва
- Доступность зала

### Множественное создание

- Можно выбрать несколько залов
- Все сеансы создаются с одинаковыми параметрами:
  - Одна дата
  - Одно время
  - Одна цена
  - Один язык и формат

## Примеры использования

### Пример 1: Создание одного сеанса

1. Выберите фильм "Аватар 2" (длительность: 192 мин)
2. Выберите зал "Зал 1"
3. Выберите дату: 15 января 2024
4. Укажите время начала: 17:00
5. Время окончания автоматически: 20:12
6. Укажите цену: 300 сом
7. Нажмите "Создать 1 сеанс"

### Пример 2: Создание нескольких сеансов

1. Выберите фильм "Человек-паук" (длительность: 148 мин)
2. Выберите залы: "Зал 1", "Зал 2", "Зал 3"
3. Выберите дату: 20 января 2024
4. Укажите время начала: 19:00
5. Время окончания автоматически: 21:28
6. Укажите цену: 250 сом
7. Выберите формат: 3D
8. Нажмите "Создать 3 сеанса"

Результат: Будет создано 3 сеанса (по одному в каждом зале) с одинаковыми параметрами.

## Советы

1. **Планируйте заранее**: Календарь позволяет выбрать любой месяц и год
2. **Используйте навигацию**: Быстро переключайтесь между месяцами с помощью стрелок
3. **Кнопка "Сегодня"**: Быстро вернитесь к текущей дате
4. **Проверяйте загрузку**: Смотрите занятые времена перед выбором
5. **Учитывайте перерывы**: Помните о 20-минутном перерыве
6. **Используйте множественный выбор**: Создавайте сеансы для нескольких залов одновременно
7. **Проверяйте цены**: Убедитесь, что цена указана правильно
8. **Прошедшие даты**: Система не позволит выбрать прошедшие даты

## Возможные ошибки

### "Выбранное время недоступно"
- **Причина**: Конфликт с существующим сеансом или недостаточный перерыв
- **Решение**: Выберите другое время или другую дату

### "Зал занят в это время"
- **Причина**: В выбранном зале уже есть сеанс на это время
- **Решение**: Выберите другой зал или другое время

### "Укажите базовую цену"
- **Причина**: Не указана цена билета
- **Решение**: Введите цену в поле "Базовая цена"

## Навигация

- **Далее →**: Переход к следующему этапу
- **← Назад**: Возврат к предыдущему этапу
- **Отмена**: Возврат к списку сеансов без сохранения

## Поддержка

Если у вас возникли вопросы или проблемы, обратитесь к администратору системы.
