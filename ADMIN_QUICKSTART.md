# Быстрый старт админ-панели

## 1. Создайте Telegram бота

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot` и следуйте инструкциям
3. Сохраните токен бота (например: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`)

## 2. Получите ваш Telegram User ID

1. Откройте [@userinfobot](https://t.me/userinfobot) в Telegram
2. Отправьте любое сообщение
3. Сохраните ваш User ID (например: `123456789`)

## 3. Добавьте настройки в БД

Откройте Prisma Studio:

```bash
npm run db:studio
```

В таблице `Setting` добавьте две записи:

| key | value |
|-----|-------|
| `ADMIN_TELEGRAM_BOT_TOKEN` | Ваш токен бота |
| `ADMIN_TELEGRAM_USER_ID` | Ваш Telegram User ID |

## 4. Войдите в админ-панель

1. Откройте http://localhost:3000/admin/login
2. Введите:
   - Email: `admin@cinema.kg`
   - Пароль: `admin123`
3. Получите код в Telegram
4. Введите код на странице подтверждения
5. Готово! Вы в админ-панели 🎉

## Важно

- **Измените пароль администратора** после первого входа
- **Не начали диалог с ботом?** Найдите вашего бота в Telegram и отправьте `/start`
- **Код не приходит?** Проверьте настройки в таблице `settings`

## Безопасность

- Админы не могут заходить на страницы пользователей
- Пользователи не могут заходить на страницы админов
- Все маршруты защищены middleware
- Коды 2FA действительны 5 минут
