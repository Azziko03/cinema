# Руководство по сообщениям при создании сеансов

## Обзор улучшений

Все сообщения теперь более профессиональные, понятные и информативные. Вместо технических терминов используются понятные объяснения.

---

## 1. Валидация при проверке доступности

### ❌ Было:
```
"Заполните все обязательные поля"
```

### ✅ Стало:

**Не выбран фильм:**
```
⚠️ Пожалуйста, выберите фильм
```

**Не выбраны залы:**
```
⚠️ Пожалуйста, выберите хотя бы один зал
```

**Не указана дата:**
```
⚠️ Пожалуйста, укажите дату начала
```

**Не указано время:**
```
⚠️ Пожалуйста, укажите время для всех слотов
```

## 3. Создание сеансов

### Валидация перед созданием

**Не проверена доступность:**
```
⚠️ Пожалуйста, сначала проверьте доступность залов
```

**Нет доступных слотов:**
```
❌ Нет доступных слотов для создания сеансов
```

**Не указана цена:**
```
❌ Укажите базовую цену билета
```

---

### Результаты создания

#### Успешное создание (все созданы):
```
✓ Успешно создано 38 сеансов!
```

#### Частичное создание (были конфликты):
```
⚠️ Создано 35 из 38 сеансов. 
   3 не удалось создать из-за конфликтов.
```

#### Ошибка при создании:
```
❌ Не удалось создать сеансы. Попробуйте еще раз.
```

#### Ошибка сети:
```
❌ Произошла ошибка при создании сеансов. 
   Проверьте подключение к интернету.
```

## 4. Детали недоступных слотов

Каждый недоступный слот отображается с полной информацией:

```
┌─────────────────────────────────────────────────┐
│ 1  🏛️ Зал 1                          [Занят]  │
│    📅 16.05.2026  🕐 14:00                     │
└─────────────────────────────────────────────────┘
```

**Элементы:**
- **Номер** - порядковый номер в списке
- **Иконка зала** 🏛️ - визуальный индикатор
- **Название зала** - понятное имя
- **Статус** - красный бейдж "Занят"
- **Дата** 📅 - в формате DD.MM.YYYY
- **Время** 🕐 - в формате HH:MM

---

## 5. Цветовая схема

### Статусы:
- 🟢 **Зеленый** - Успех, все доступно
- 🔵 **Синий** - Информация, частичная доступность
- 🟡 **Оранжевый** - Предупреждение, недоступные слоты
- 🔴 **Красный** - Ошибка, нет доступных слотов

### Иконки:
- ✓ - Успех
- ℹ️ - Информация
- ⚠️ - Предупреждение
- ❌ - Ошибка
- 🏛️ - Зал
- 📅 - Дата
- 🕐 - Время

## 7. Преимущества новых сообщений

### Понятность ✓
- Вместо "Конфликт расписания" → "Зал занят в это время"
- Вместо "Ошибка" → Конкретное объяснение проблемы

### Профессионализм ✓
- Структурированная информация
- Визуальные индикаторы (иконки, цвета)
- Четкие инструкции к действию

### Информативность ✓
- Показывается не только что не так, но и почему
- Даются рекомендации по исправлению
- Детальная информация о каждом конфликте

### Удобство ✓
- Легко сканировать глазами
- Быстро понять суть проблемы
- Понятно что делать дальше

## 8. Технические детали

### Toast-сообщения
```typescript
// Типы toast-сообщений:
showToast(message, "success");  // Зеленый, галочка
showToast(message, "info");     // Синий, информация
showToast(message, "warning");  // Оранжевый, предупреждение
showToast(message, "error");    // Красный, ошибка
```

### Визуальные блоки
```typescript
// Цветовые схемы:
bg-green-500/10 border-green-500/30  // Успех
bg-blue-500/10 border-blue-500/30    // Информация
bg-orange-500/5 border-orange-500/30 // Предупреждение
bg-red-500/10 border-red-500/30      // Ошибка
```
